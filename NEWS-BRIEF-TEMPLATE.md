# 新闻简报模板规范

## 📋 内容结构

### 1. 头部 (Header)
```
📰 每日新闻简报
2026 年 3 月 7 日 · 星期六
```
- 标题：带 emoji，居中
- 日期：显示年月日 + 星期

### 2. 新闻条目 (每条新闻)
```
[编号圆圈] 新闻标题
新闻内容段落（2-3 句话，事实描述）

┌─────────────────────────────┐
│ 🔍 AI 深度解析               │
│ AI 分析内容（2 句话，深度解读）│
└─────────────────────────────┘
```

**每条新闻必须包含**：
1. **编号圆圈** - 渐变紫色背景，白色数字
2. **新闻标题** - 一句话概括核心事件
3. **新闻内容** - 2-3 句事实描述（谁、做了什么、结果）
4. **AI 深度解析** - 2 句深度分析（影响、意义、趋势）

### 3. 页脚 (Footer)
```
亚连观察者 · 每日新闻简报
数据来源：Exa 搜索 · AI 整理分析
https://ycnick123.github.io/yanlian-observer/
```

## 📐 样式规范

### 容器
- 宽度：max-width 600px
- 背景：白色
- 圆角：16px
- 阴影：0 20px 60px rgba(0,0,0,0.3)
- 内边距：40px

### 标题区
- 主标题：32px，字重 900，黑色
- 日期：18px，字重 500，灰色
- 底部分隔线：3px  solid #667eea

### 新闻条目
- 间距：32px
- 底部分隔线：1px solid #eee（最后一条无分隔线）

### 编号圆圈
- 尺寸：32px × 32px
- 背景：渐变 (#667eea → #764ba2)
- 数字：白色，16px，字重 700
- 圆角：50%

### 新闻标题
- 字号：20px
- 字重：700
- 颜色：黑色
- 行高：1.4

### 新闻内容
- 字号：18px
- 字重：500
- 颜色：#333
- 行高：1.6

### AI 深度解析框
- 背景：#fef3c7（黄色）
- 左边框：4px solid #f59e0b（橙色）
- 圆角：8px
- 内边距：16px
- 标签：14px，字重 700，#92400e
- 内容：16px，字重 500，#78350f，行高 1.5

### 页脚
- 居中对齐
- 字号：14px
- 颜色：灰色
- 顶部分隔线：2px solid #eee

## 📝 内容要求

### 新闻选择标准
1. **6 条新闻** - 国内 3 条 + 国际 3 条
2. **重要性** - 政治、经济、科技、国际关系优先
3. **时效性** - 24 小时内最新热点

### 新闻内容格式
```
[人物/机构] + [动作/事件] + [结果/影响]

示例：
国务院总理李强在政府工作报告中宣布，2026 年中国 GDP 增长目标设定为 4.5% 至 5%，这是自 1991 年以来首次低于 5% 门槛。
```

### AI 解析格式
```
[现象解读] + [深层意义/未来影响]

示例：
GDP 目标下调反映政府务实态度，承认经济增速放缓现实，为结构性改革留出空间。"十五五"规划将是中国式现代化最关键五年。
```

## 🎯 VC 投资简报特别说明

VC 简报使用**完全相同**的格式，仅标题改为：
```
📈 每日 VC 投资简报
```

内容结构：
1. 公司名 + 融资轮次 + 金额
2. 投资方
3. 业务简介
4. AI 解析（行业趋势/投资意义）
